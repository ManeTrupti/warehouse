# Planning & Scheduling (PPS) UI

Multi-tenant Production Planning & Scheduling frontend (React, Vite, Redux Toolkit, React Router, Tailwind).

## Quick start

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000). For path-based tenant strategy, use URLs like:

- `http://localhost:3000/tenant/tenant1/dashboard`
- `http://localhost:3000/tenant/tenant2/planning`
